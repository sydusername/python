# find_nearest_point

#### 介绍
遍历循环excel中的所有点，挨个寻找离它最近的点，并按照距离进行排序

#### 软件架构
软件架构说明


#### 安装教程

1.  python版本3.10.13
2.  安装的库pywin32
3.  需要电脑安装excel或wps

#### 使用说明

1.  修改py文件中point_xy_path变量，值为excel文件的位置和名称
2.  选择包含excel的文件，第一行为表头，从第二行开始为数据
3.  生成的excel文件与原文件在同一文件夹下
